How to Use the Web Application

1. Upload the Questions File

You need to upload an Excel file containing the questions in the following format:

Pregunta            Respuesta A        Respuesta B           Respuesta C              Respuesta D

Number + Question	 a) Possible answer	 b) Posible answer	   c Posible answer         Optional


Example:

Pregunta	                           Respuesta A	         Respuesta B	    Respuesta C	    Respuesta D
1.- What is the capital of France?	a) Berlin	             b Madrid	        c Paris	        d Rome



2. Upload the Correct Answers File

You must upload a second Excel file that contains the correct answers in the following format:

Pregunta          Respuesta correcta
Question number   Question letter

Example:

Pregunta	Respuesta Correcta
1	        c



The first column should match the question numbers from the previous file.

The second column should contain the correct answer letter (a, b, c, or d).


3. Processing and Results

Once both files are uploaded (web will ask for answers doc at the end of answer page):

It will make different chapters based on the excel pages, then make subsets of 20 questions

When you confirm the test it will notify how many answers the user chose were correct, and then show what the correct answer was.

Notes

Ensure both Excel files follow the expected structure to avoid errors.

The question numbers must be unique and consistent between both files.

The web application supports multiple-choice questions with three or four options (a, b, c, d).
